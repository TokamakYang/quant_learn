# Quant Learn

量化交易学习项目

## 项目描述

这是一个量化交易学习项目，包含了使用VectorBT进行量化策略开发与回测的实践内容。

## 文件说明


- `vectorbt_learn_rewrite.html` - VectorBT学习笔记（HTML版本）


## 技术栈

- Python
- VectorBT
- Jupyter Notebook
- 量化交易策略开发

## 使用方法

1. 安装依赖：
   ```bash
   pip install vectorbt jupyter pandas numpy
   ```

2. 启动Jupyter Notebook：
   ```bash
   jupyter notebook
   ```

3. 打开 `vectorbt_learn_rewrite.ipynb` 开始学习

## 学习内容

- VectorBT基础使用
- 布林线策略开发
- 量化策略回测
- 策略性能评估
